Budget · decay · crowding · saturation

Discipline Rule Boundary
Budget max 12 selected refs per actor / period finite, deterministic
Per-dimension cap 3 refs / focus dimension prevents single-axis dominance
Decay inherited focus weight halves each period without reinforcement integer-counted, never stochastic
Crowding new high-weight focus displaces lowest-weight inherited focus deterministic priority order
Saturation above 8 focus labels → drop_oldest cap on simultaneous attention
None of these is a calibrated probability. None of these is a behavioral forecast. They are bookkeeping rules on a finite attention object.

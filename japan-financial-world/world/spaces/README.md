# Spaces

Spaces are bounded parts of the world. They should not mutate each other
directly.

Required spaces:

- `corporate/`
- `investor/`
- `bank_debt/`
- `exchange/`
- `real_estate/`
- `information/`
- `policy/`
- `external_macro/`

Allowed cross-space channels:

- Asset ownership
- Contracts
- Market prices
- Information signals
- Constraints
